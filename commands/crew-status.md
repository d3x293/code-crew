---
description: "Show CodeCrew status - team config, index health, skill catalog, and task history"
---

Invoke the **codecrew:team-status** skill to display:
- Project profile summary
- Active agent team and their roles/models
- Codebase index health (last indexed, file count, staleness)
- Skill catalog stats (total available, stack-specific, active)
- Recent task history if available
