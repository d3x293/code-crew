---
description: "Configure CodeCrew settings - model tiers, default mode, token budgets, and more. Subcommands: show, set, reset."
---

Invoke the **codecrew:crew-config** skill with the user's config subcommand.

Parse the subcommand:
- `/crew config show` → Display current configuration
- `/crew config set <key> <value>` → Update a setting
- `/crew config reset` → Restore all defaults

Pass the full subcommand and arguments to the skill.
